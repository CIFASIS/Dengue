# Modelos de simulación de casos de dengue.

Este repositorio consiste del trabajo realizado en el **CIFASIS** para el modelado de casos de dengue.

## Estructura

Se tienen dos carpetas:

 * **Datos**: Aquí se encuentran los datos utilizados, tanto meteorológicos como de infecciones del virus.
 * **Modelos**: Esta carpeta contiene todos los modelos que se fueron creando, no únicamente con el objetivo de resolver
 la problemática, si no también para ver las distintas formas de cada moddelo particular.
